How to use WixMediaImage in django

  • Add wixmedia-django to INSTALLED_APPS
  • Add WIXMEDIA_ROOT to settings. 
    example:
      WIXMEDIA_ROOT = 'http://endpoint.com'
  • Add 
          {% load wixmediatags %}
    to template
  • Use tag image_url in template.
  • See example in example_django.html file.
  • See it working.

How to use WixMediaImage in jinja

  • Add image_url helper to jinja2 environment 
          JINJA_ENVIRONMENT.globals['image_url'] = image_url
  • Use image_url in template.
  • See example in example_jinja2.html file.
  • See it working.
